# Retail Sales Performance Analytics Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze retail sales performance, profitability, and regional trends. The dashboard enables users to monitor key business metrics and identify actionable insights through interactive visualizations.

## Tools Used

- Power BI
- DAX
- Power Query
- Data Visualization

## Key KPIs

- Total Sales: ₹8.10M
- Total Profit: ₹1.02M
- Total Orders: 7.99K

## Dashboard Features

- Monthly Sales Trend Analysis
- Regional Profit Analysis
- Category-wise Sales Analysis
- State-wise Sales Performance
- Interactive Slicers and Filters

## Business Insights

- Florida emerged as the top-performing state by sales.
- South Region generated the highest profit.
- Technology and Office Supplies contributed significantly to revenue.
- Sales peaked during August and December.
